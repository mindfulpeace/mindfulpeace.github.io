# 第一单元 入门篇

## 单元概述

通过《认识静心学堂》的学习，我们认识了静心学堂的缘起、特色、课程以及修学方式和方法，明确了为什么要在静心学堂学习佛法。

通过对信仰、传统文化以及人生问题的思考，帮助我们正确认识信仰，了解佛教是中国文化不可或缺的重要组成部分，了解佛法要解决的是我们的人生大事，了解佛法解决人类永恒问题的智慧。
