# 佛法 与 解脱心理学

佛法：心性之学，立足解脱

- 解脱是佛法修行的核心，无论大乘小乘，根基悟性
- 修行关键在于正确用心，凡夫心贼子，要认得识破
- 解脱意为：发出离心，解脱轮回！
