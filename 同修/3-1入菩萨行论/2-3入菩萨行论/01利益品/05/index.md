# 第 5 课 利益品 第五课

## 【修学内容】

[《入菩萨行论》讲记:第九、十讲](text)

## 【学前思考】

一、上课已教证了菩提心的利益，及理证了愿菩提心的利益。
本课如何理证行菩提心的利益呢？

二、为什么要成为菩萨呢？
菩萨又为什么伟大呢？
对菩萨的不同态度对自己又有什么影响呢？
应该如何对待菩萨呢？

三、为什么要对菩提心生起依赖感呢？
如何才能生起这份依赖感呢？

四、菩提心是否关起门来修就可以了呢？
修行能离开座下修吗？
为什么呢？

## 【修学标准】

一、认真观听教材三遍以上。

二、理解该部分内容的意思，列出简要提纲。

三、搞清楚这部分内容要说明的问题及论证方式，并反复思惟。

## 【修学思考】

一、愿菩提心的利益无量无边，而行菩提心的利益比愿菩提心更加广大。

(一)上课已证明了愿菩提心的利益，本课更进一步引《月灯经》来说明愿菩提心的利益之大。

而行菩提心的利益要大于愿菩提心，行菩提心的功德自然更是无量无边。
对法师的开示，我理解了吗？
我愿意接受吗？

(二)众生有没有能力离苦得乐呢？
众生的现状是如何的？
对法师的开示我理解了吗？
我自己的现状是怎么样的？
与这里阐述众生的现状不同吗？
如果不学佛法，我有能力离苦得乐吗？
菩萨的工作是什么？
菩萨的行为有价值吗？
菩萨的工作与众生的需要吻合吗？
这样的论证能强有力地证明行菩提心的利益吗？
我对菩提心的利益生起希求之心了吗？

二、法师开示要推广成为菩萨的理念，为什么呢？
如何成为菩萨？
又如何对菩萨进行重新认识呢？
我想成为菩萨吗？
有决心了吗？

三、菩萨真是伟大，为什么呢？

(一)世间上知恩图报的人已经了不起了，那么菩萨又是怎样的呢？
通过这样的对比是否可以显示菩萨的伟大呢？
我对菩萨的伟大的认识如何？
能接受这样的对比显示吗？

(二)世间的善人应受尊重，相比之下，菩萨又是如何的呢？
如此伟大的菩萨，我是否愿意以他们为榜样呢？
作为自己学习的榜样呢？

(三)对一个修养有素的菩萨来说，即使遭遇磨难，反而使菩提心的力量更加强大。
对于具备这样伟大人格的菩萨，我生起了礼敬之心吗？
对于这种人格的源头——菩提心，我是否也生起了皈敬之心呢？
四、通过《菩提心利益品》的学习，建立起了对菩提心的强大信心了吗？
如何来建立这样的信心呢？

(一)法师要求我们养成读诵和背诵的习惯。
我愿意接受法师的说法吗？
我在读诵或背诵《入行论》了吗？

(二)法师开示要通过学本品对菩提心生起深深的依赖感。
我接受这一观点吗？
我对菩提心生起了深深的依赖感了吗？
法师开示要如何来生起这种依赖感呢？
我理解法师的开示吗？
我接受法师的开示吗？
我愿意遵照执行吗？
如果与八步骤相合起来进行思考，有何启发？

五、菩提心是否关起门来修就可以了呢？
修行能离开座下修吗？
为什么呢？
我理解了法师对此所作的开示了吗？
我接受法师的说法吗？

## 【修学检验】

一、行菩提心的利益广大无边，这样的认识在内心建立起来了吗？

二、菩萨的人格真是伟大!认同这样的观念，并对菩萨、对菩提心生起皈敬之心了吗？

三、要成为菩萨，对菩提心生起深深的依赖感。
这样的观念树立起来了吗？

四、法师开示要养成读诵和背诵的习惯，用观察修和安住修不断反复地修菩提心，使学习的过程成为发菩提心的过程。
这样的观念建立起来了吗？
我开始这样学习了吗？
