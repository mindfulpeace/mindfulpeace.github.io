# 静心学堂课程

_随喜师兄一起维护课程 MD 文本。_

## 项目介绍

为静心学堂学员提供课程"纯"文本查询服务

技术栈： github pages & jekyll & markdown

部署链接： http://mindfulpeace.github.io
